**NETFLIX SNIPER**는 넷플릭스의 방대한 콘텐츠 중 무엇을 볼지 결정하지 못하는 '결정 장애'를 해결하기 위해 제작된 **고성능 저격용 추천 앱**입니다. 2000년대 초반의 Y2K 감성과 현대적인 퀀트 스코어링(평점 분석) 시스템이 결합되었습니다.

---

## 🕹️ 핵심 기능 (Core Features)

1. **장르별 정밀 저격 (Genre Filtering)**:
   - 영화, 드라마, 예능, 애니메이션 중 원하는 장르를 선택하여 범위를 좁힐 수 있습니다.
2. **실패 없는 명작 리스트**:
   - 네이버 평점 8.5~9점대의 검증된 작품들만 데이터베이스에 탑재되어 있습니다.
3. **중복 저격 방지 (Anti-Repeat Logic)**:
   - 바로 직전에 나온 결과가 다시 나오지 않도록 필터링하여 매번 신선한 추천을 제공합니다.
4. **Y2K 멀티미디어 체험**:
   - 세기말 감성의 8비트 사운드 효과와 화려한 전광판 효과를 제공합니다.
5. **모바일 최적화**:
   - iOS/Android 홈 화면에 추가하여 네이티브 앱처럼 사용할 수 있습니다.

---

## 🚀 시작하기 (Quick Start)

1. `index.html` 파일을 다운로드합니다.
2. 브라우저(Chrome, Safari 등)에서 파일을 엽니다.
3. 원하는 장르를 선택합니다.
4. **[ SNIPE IT! ]** 버튼을 눌러 운명의 작품을 확인합니다.

---

## 📱 모바일 앱으로 설치 (iOS/Android)

- **iOS**: Safari에서 접속 -> 하단 [공유] 버튼 -> **[홈 화면에 추가]**
- **Android**: Chrome에서 접속 -> 상단 [더보기] 버튼 -> **[홈 화면에 추가]**

## 🛠️ 기술 사양 (Specs)

- **Frontend**: HTML5, CSS3 (Win98 UI style)
- **Logic**: Vanilla JavaScript
- **Audio**: Web Audio API (Retro Sound Engine)
- **Design**: Share Tech Mono & VT323 Google Fonts
